Что-бы построить сетку блоков мы можем использовать свойство float со значением left для касса .rows. Так-же добавим элемент div с классом .clear в самый конец контейнера .parent и придадим классу .clear свойство clear со значением both как и во втором задании. Так у нас получится сетка блоков 4х3. Стоит заметить что блоки .row занянимают всё пространство по ширине класса .parent, но не выходят из него. Что-бы сделать например сетку 2х6 или другую нам нужно будет просто изменить значение в свойстве width класса .parent.
